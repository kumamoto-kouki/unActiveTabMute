# unActiveTabMute

非アクティブなタブの音声を自動でミュートする Google Chrome 拡張機能です。

## 概要

複数のタブを開いているとき、今見ていないタブから音声が流れ続けてしまう問題を解決します。フォーカスされているウィンドウのアクティブタブのみ音声を許可し、それ以外のタブは自動的にミュートします。

## 機能

- タブを切り替えると、前のタブが自動でミュートされる
- 複数ウィンドウに対応（フォーカスされているウィンドウのアクティブタブのみ再生）
- 拡張機能アイコンをクリックするだけで機能のON/OFFを切り替え可能
- OFF時はアイコンがグレーに変わり、状態が一目でわかる
- Chrome 再起動後もON/OFF状態を維持

## インストール（開発者モード）

1. このリポジトリをクローンまたはダウンロードする
   ```
   git clone <repository-url>
   ```

2. Chrome で `chrome://extensions/` を開く

3. 右上の「デベロッパーモード」をオンにする

4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このフォルダを選択する

## 使い方

インストール後は自動で動作します。

| 操作 | 動作 |
|------|------|
| タブを切り替える | 前のタブが自動でミュートされる |
| 別ウィンドウに切り替える | 元のウィンドウのタブがミュートされる |
| アイコンをクリック（青） | 機能をOFFにする（全タブのミュート解除） |
| アイコンをクリック（グレー） | 機能をONにする（ミュート制御を再開） |

## ファイル構成

```
unActiveTabMute/
├── manifest.json          # Chrome拡張機能マニフェスト（MV3）
├── background.js          # サービスワーカー（ミュート制御のコアロジック）
├── icons/
│   ├── icon16.png         # ツールバーアイコン（有効時）
│   ├── icon48.png         # 拡張機能管理画面アイコン（有効時）
│   ├── icon128.png        # Chrome Web Store アイコン（有効時）
│   ├── icon16_off.png     # ツールバーアイコン（無効時）
│   ├── icon48_off.png     # 拡張機能管理画面アイコン（無効時）
│   └── icon128_off.png    # Chrome Web Store アイコン（無効時）
└── scripts/
    └── create_icons.js    # アイコンPNG生成スクリプト（開発用）
```

## 技術仕様

- **Manifest Version**: 3
- **対応ブラウザ**: Chrome 112 以上、Chromium ベースの MV3 対応ブラウザ
- **権限**:
  - `tabs` — タブ情報の読み取りとミュート制御
  - `windows` — ウィンドウフォーカス変更の検知
  - `storage` — ON/OFF状態の永続化
- **外部依存関係**: なし
- **データ送信**: なし（全処理はローカルで完結）

## アイコンの再生成

アイコンを変更したい場合は、`scripts/create_icons.js` を編集して以下を実行してください。

```bash
node scripts/create_icons.js
```
