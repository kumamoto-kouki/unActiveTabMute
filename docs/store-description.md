## 【日本語】詳細説明

```
複数のタブを開いていると、いつの間にか別のタブから音楽や動画の音声が流れ出してしまうことはありませんか？

unActiveTabMute は、今あなたが見ているタブ以外の音声をすべて自動でミュートする Chrome 拡張機能です。タブを切り替えるだけで、前のタブは静かになり、新しいタブの音声が再生されます。

【主な機能】
・非アクティブなタブの音声を自動ミュート
・複数ウィンドウに対応（フォーカスしているウィンドウのアクティブタブのみ再生）
・拡張機能アイコンをクリックするだけで機能のON/OFF切り替え
・OFF 時はアイコンがグレーに変わり、状態が一目でわかる
・Chrome 再起動後もON/OFF の設定を記憶

【プライバシーについて】
この拡張機能は、タブのミュート制御のみを行います。閲覧データや個人情報を収集・送信することは一切ありません。すべての処理はお使いのブラウザ内で完結します。
```

## 【English】Detailed Description

```
Do you find yourself distracted by audio from background tabs — a video you forgot about, a music player left open, or an auto-playing ad?

unActiveTabMute automatically mutes every tab except the one you're actively viewing. Switch tabs, and the previous one goes silent instantly. No manual muting required.

Features:
- Automatically mutes all inactive tabs
- Multi-window support — only the active tab in your focused window plays audio
- One-click toggle: click the extension icon to enable or disable
- Icon turns gray when disabled so you always know the current state
- Remembers your on/off preference across Chrome restarts

Privacy:
This extension only controls tab muting. It does not collect, store, or transmit any browsing data or personal information. All processing happens locally in your browser.
```
