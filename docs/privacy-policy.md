# Privacy Policy

Last updated: 2026-03-24 | Extension: unActiveTabMute

unActiveTabMute ("the Extension") is a Google Chrome extension that automatically mutes audio on inactive browser tabs. This page explains what data the Extension accesses and how it is handled.

## Data Collection

The Extension does **not** collect, store, or transmit any personal information or user data. Specifically, the Extension does not collect:

- Browsing history or visited URLs
- Tab titles or page content
- Personal or identifying information
- Usage statistics or analytics

## Local Storage

The Extension stores only one setting locally on your device using `chrome.storage.local`: whether the muting feature is enabled or disabled. This value never leaves your browser.

## Permissions Used

- **tabs** — Read and update the mute state of browser tabs.
- **windows** — Detect window focus changes to correctly apply muting across multiple windows.
- **storage** — Save the on/off preference locally so it persists across browser restarts.

No permission is used to access page content, browsing history, or any personal data.

## Third-Party Services

The Extension does not communicate with any external server or third-party service. All processing occurs entirely within your browser.

## Remote Code

The Extension does not load or execute any code from remote sources.

## Changes to This Policy

If this policy is updated, the "Last updated" date at the top of this page will be revised. Continued use of the Extension after any changes constitutes acceptance of the updated policy.

## Contact

For questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/kumamoto-kouki/unActiveTabMute).

# プライバシーポリシー

最終更新日: 2026年3月24日 | 拡張機能: unActiveTabMute

unActiveTabMute（以下「本拡張機能」）は、非アクティブなブラウザタブの音声を自動でミュートする Google Chrome 拡張機能です。本ページでは、本拡張機能がアクセスするデータとその取り扱いについて説明します。

## データの収集について

本拡張機能は、個人情報およびユーザーデータを一切収集・保存・送信しません。具体的には、以下の情報を収集しません。

- 閲覧履歴・アクセスした URL
- タブのタイトルやページの内容
- 個人を特定できる情報
- 利用統計・アナリティクス情報

## ローカルへの保存

本拡張機能は、ミュート機能の有効・無効の設定のみを `chrome.storage.local` を使ってお使いのデバイス上にローカル保存します。この情報がブラウザの外部に送信されることはありません。

## 使用する権限

- **tabs** — タブのミュート状態の読み取りおよび変更のために使用します。
- **windows** — 複数ウィンドウ間の切り替えを検知し、正しくミュートを制御するために使用します。
- **storage** — ON/OFF 設定をローカルに保存し、ブラウザ再起動後も設定を維持するために使用します。

いずれの権限も、ページの内容・閲覧履歴・個人情報へのアクセスには使用しません。

## 外部サービスとの通信

本拡張機能は外部サーバーやサードパーティサービスと一切通信しません。すべての処理はお使いのブラウザ内で完結します。

## リモートコード

本拡張機能は外部からコードを読み込んで実行することはありません。

## 本ポリシーの変更

本ポリシーを更新した場合は、ページ上部の最終更新日を変更します。更新後も本拡張機能を継続して使用することで、変更後のポリシーに同意したものとみなします。

## お問い合わせ

本プライバシーポリシーに関するご質問は、[GitHub リポジトリ](https://github.com/kumamoto-kouki/unActiveTabMute) の Issue よりお問い合わせください。
